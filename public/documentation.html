myFlix Documentation

This application will allow users to access information regarding movies, directors and other movie relating information. This includes sorting and creating personal list of their personal favorites.

Endpoint HTTP Method Description
/movies GET Returns all movies
/users GET Returns all Users
/movies/:Title GET Returns a movie based on name given
/movies/genres/:genreName GET Returns movie that have specific genre
/movies/directors/:directorName GET Returns data from a provided director name
/users POST Allows new users to registers
/users/:Username PUT allows users to update their information
/users/:Username/movies/:MovieID POST allows users to add movies to their favorites list
/users/:Username/movies/:MovieID DELETE allows users to delete movies from their favorites list
/users/:Username DELETE allows users to deregister
/ GET returns simple message
