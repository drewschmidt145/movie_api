myFlix Documentation

This application will allow users to access information regarding movies, directors and other movie relating information. This includes sorting and creating personal list of their personal favorites.

Endpoint HTTP Method Description Request Response
/movies GET Returns all movies N/A Return a List of all movies including data referring to them
/users GET Returns all Users N/A Return a list of all users, including all information on their account
/movies/:Title GET Returns a movie based on name given N/A Return a single movie based on name, will also give data referring to that movie
/movies/genres/:genreName GET Returns movie that have specific genre N/A Returns list of movies that have been given that specific genre
/movies/directors/:directorName GET Returns data from a provided director name N/A Returns list of movies that specific director is attached to
/users POST Allows new users to registers A JSON object with the following data: { "Username": "Example1" "Password": "examplepassword1" "Email": "example@gmail.com" "Birthday": "08-09-2000" } Returns the same object given in request, but it includes a generated _ID and __v version number attached
/users/:Username PUT allows users to update their information A JSON object with the following data updated: { "Username": "Example1" "Password": "examplepassword1" "Email": "example@gmail.com" "Birthday": "08-09-2000" } Returns the users information with all updated fields
/users/:Username/movies/:MovieID POST allows users to add movies to their favorites list N/A Returns Users information, including newly updated favorites list
/users/:Username/movies/:MovieID DELETE allows users to delete movies from their favorites list N/A Returns Users information, including newly updated favorites list
/users/:Username DELETE allows users to deregister N/A Returns a message verifying that the user has been deleted
