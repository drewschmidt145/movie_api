myFlix Documentation

This application will allow users to access information regarding movies, directors and other movie relating information. This includes sorting and creating personal list of their personal favorites.

Business Logic URL HTTP Method Path Variables Request body data format in JSON Response body data format in JSON
Return a list of all movies to the user /movies GET None None A JSON object holding an array with all movies.
Return data of a single movie to user /movies/:Title GET Title: Inception None A JSON object that contains data of a single movie. This includes description, genre, director, image URL, and whether it is a featured or not, Example: { "Title": "Inception", "Description": "A thief who steals corporate secrets...", "Genre": "Action", "Director": "Christopher Nolan", "ImageURl": "https://en.wikipedia.org/wiki/Inception#/media/File:Inception_(2010)_theatrical_poster.jpg", "Feature Film": "true" }
